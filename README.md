~/.vimrc
```
" Set tab width to 2 spaces
set tabstop=2
set shiftwidth=2
set expandtab

" Enable auto-indentation
set autoindent
set smartindent

" Show existing tab characters as `▸·`
set list
set listchars=tab:▸·,trail:·

" Enable line numbers
set number
```
